## * 查询币种价格 `/trade/getCoinPrice`

- **请求方式** `POST`

- **请求参数**

| 参数      | 类型   |             是否必填             | 最大长度 | 描述              | 示例值                    |
| :-------- | ------ | :------------------------------: | :------: | ----------------- | :------------------------ |
| orderType | String | <font color=#FF4D4D>true</font>  |    32    | 支付类型          | 可用值:BY_MONEY,BY_AMOUNT |
| coinName  | String | <font color=#54FF9F>false</font> |    32    | 支付币种          | USDT                      |
| amount    | String | <font color=#54FF9F>false</font> |    32    | 支付数量          | 0.01                      |
| currency  | String | <font color=#54FF9F>false</font> |    32    | 3 位 ISO 货币代码 | CNY                       |
| money     | String | <font color=#54FF9F>false</font> |    32    | 支付金额          | 100                       |

- **请求示例**

```json
{
    "orderType": "BY_AMOUNT",
    "coinName": "USDT",
    "amount": "100",
    "currency": "CNY",
    "money": "100"
}
```

- **返回参数**

| 参数     | 类型   | 描述              | 示例值                    |
| :------- | ------ | ----------------- | :------------------------ |
| payType  | String | 支付类型          | 可用值:BY_MONEY,BY_AMOUNT |
| coinName | String | 支付币种          | USDT                      |
| amount   | String | 总数量            | 0.01                      |
| currency | String | 3 位 ISO 货币代码 | CNY                       |
| money    | String | 总金额            | 100                       |
| price    | String | 价格              | 100                       |

- **返回示例**

```json
{
    "payType": "BY_AMOUNT",
    "coinName": "USDT",
    "amount": "100",
    "currency": "CNY",
    "money": "100",
    "price": "100"
}
```

| ------- | :----: | :-----------------------------: | -------- | :------ |
| code    | String | <font color=#FF4D4D>true</font> | 响应code | 200     |
| message | String | <font color=#FF4D4D>true</font> | 响应信息 | success |

- **响应示例**

```json
{
    "code": 200,
    "message": "success"
}
```
